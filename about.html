About the Project

This website was created as part of a learning activity to understand how Git and GitHub work in managing and collaborating on projects. The project demonstrates basic web development using HTML while applying version control using Git and GitHub.

Objectives

  • Learn basic HTML webpage creation
  • Practice collaboration using GitHub
  • Understand version control using Git

Technologies Used

HTML, Git, and GitHub.
